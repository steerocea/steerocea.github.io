Cellar


About

Cellar is 3D dungeon crawler that I developed using the Unreal engine. As a proud member of a skilled team of three, we created this as our final project for second year game devleopment course.


The core game mechanic revolves around the cycle of exploration and progression. Players begin their journey equipped with basic gear and venture into procedurally generated dungeons, engaging in battles against various enemies and uncovering better gear hidden within chests.


This game's creation really honed my ability to use Git as it was my first long term coding project where amongst the team, Git was the main hub for our development.

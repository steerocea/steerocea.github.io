Deckay


About

Deckay is a card game developed using the Unity engine. In a team of 6 as a part of the Game Jam Aotearoa 2023, we built the game under a 48-hour time limit.


This game's creation was my introduction to using Unity as well as the C# language, greatly improving my skillset.
